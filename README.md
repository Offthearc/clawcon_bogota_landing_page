# ClawCon Bogota Landing Page

Landing page for ClawCon Bogota — Colombia's premier tabletop gaming convention.

---

## Prerequisites

- Node.js 18+ (tested on v26)
- npm 9+

---

## Setup

```bash
git clone git@github.com:Offthearc/clawcon_bogota_landing_page.git
cd clawcon_bogota_landing_page
npm install
```

---

## Run dev server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## Run tests

```bash
npm test
```

---

## Lint

```bash
npm run lint
```

---

## Type check

```bash
npm run typecheck
```

---

## Production build

```bash
npm run build
```
