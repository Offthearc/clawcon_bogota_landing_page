# ClawCon Bogota Landing Page

Landing page for ClawCon Bogota — a tabletop gaming convention in Bogota, Colombia.

---

## Prerequisites

- **Node.js** >= 20
- **npm** >= 10

---

## Setup

```bash
git clone git@github.com:Offthearc/clawcon_bogota_landing_page.git
cd clawcon_bogota_landing_page
npm install
```

---

## Dev server

```bash
npm run dev
```

Opens at `http://localhost:5173`.

---

## Build

```bash
npm run build
```

Output is in `dist/`.

---

## Tests

```bash
npm test
```

---

## Lint

```bash
npm run lint
```

---

## Design tokens

```bash
npm run design:tokens   # regenerate src/theme/tokens.css from DESIGN.md
npm run design:check    # verify tokens are in sync
```
